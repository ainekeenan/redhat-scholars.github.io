== Introduction to OpenShift Streams for Apache Kafka

[cols="1,3,2"]  
|===
| *Duration* | *Audience* | *Level*
|1 hour
|Developers that want to learn the basics of Apache Kafka.
|Intermediate
|===

In this tutorial you will:

* Create a Kafka instance using OpenShift Streams for Apache Kafka.
* Manage access using Service Accounts and RBAC.
* Create a topic in the Kafka instance.
* Produce and consume messages using CLI tools.
* Use the Red Hat OpenShift Application Services (RHOAS) Operator and CLI to connect the OpenShift Streams for Apache Kafka to an OpenShift cluster.
* Bind a Quarkus application to OpenShift Streams for Apache Kafka using the Red Hat OpenShift Application Services CLI.
