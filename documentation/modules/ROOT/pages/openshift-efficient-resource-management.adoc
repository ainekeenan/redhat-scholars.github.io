Let’s explore best practices and useful settings to manage application’s resources in an OpenShift cluster. Since the load of cluster usage may increase rapidly due to traffic load and the amount of applications deployed, it’s better to set up resource quotas to keep the workloads running smoothly. +
 +


image::https://redhat-scholars.github.io/openshift-apps-resource-management/openshift-apps-resource-management/_images/os-overview-image.png["OpenShift"]

What you will learn:

* Bootstrap project
* Adapt configurations
* Define custom health checks
* Deploy to OpenShift
* Adjust resource quotas
* Test configurations
* Tailor metrics
* Sizing resource limits
